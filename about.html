About SleepAI

An AI-powered sleep disorder detection system built using machine learning, clinical data, and wearable sensor integration.

Project Overview

SleepAI is a full-stack machine learning web application designed to detect sleep disorders from patient vitals, lifestyle metrics, clinical markers, and wearable sensor readings. The system uses an XGBoost classifier trained on 1,000 patients with realistic medical patterns, achieving 98.5% accuracy across 5 disorder categories.

The wearable sensor columns are designed to be swapped with real data from devices like Fitbit, Apple Watch, Oura Ring, or custom IoT sensors — making this system future-ready for real clinical deployment.
